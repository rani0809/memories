# Memories
### A Social Media Web App built using MERN stack  


### Current Features
- Registration and login using a secure Authentication System
- Google Sign in using OAuth Client
- Create, Like, Delete, Update Posts

## Libraries Used : 
### For Frontend
- React (Javascript Framework)
- Redux
- Material UI
- Axios
- React Router
- react-google-login
- jwt-decode

### For Backend
- Nodejs
- Express JS
- Mongoose
- bcryptjs

### Database
- MongoDB
